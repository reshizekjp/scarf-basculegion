# scarf-basculegion

ポケモンチャンピオンズ（シーズンM-3・シングル）用 最速スカーフイダイトウのダメージ早見表。

- **早見表**: https://reshizekjp.github.io/scarf-basculegion/
- **ポケモンデータ**: https://reshizekjp.github.io/scarf-basculegion/details.html

## 機能

- おはかまいり／ウェーブタックル／アクアジェット／クイックターンの4技切替
- 使用率TOP235・メガ形態対応・主流型／高耐久型の耐久想定切替
- 相手パーティ6体選択（対戦モード）・S先手判定（スカーフS214基準）
- 全ポケモンの詳細データ（技・もちもの・特性・性格・配分・パートナー）

## 更新方法

`index.html` と `details.html` を上書きアップロードするだけ（GitHub Actionsが自動デプロイ）。
